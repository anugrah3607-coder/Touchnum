TOUCHNUM

Aktifkan TOUCHNUM dengan ctrl+alt+esc.
Memori aktif: 1
Terakhir: -
• Sentuhan pada touchpad akan langsung mengirim aksi ke Windows.
• Klik tombol UI untuk mengganti label dan aksinya (SendKeys atau LaunchApp).
• Contoh SendKeys: 7, {ENTER}, {BACKSPACE}, ^c (Ctrl+C), ^v, dll.
• Contoh LaunchApp: menuliskan alamat aplikasi di file explorer {C:\Windows\System32\(nama aplikasi).exe}
